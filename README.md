#school-chime

研究室でチャイムを鳴らしたい時に。


#Usage
Linux or OS X

```
$ git clone git@github.com:FromAtom/school-chime.git
$ cd school-chime.git
$ bundle
$ whenever --update-crontab
```

##Linux
require `mpg123`

```
$ yum install mpg123
```
